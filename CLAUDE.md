## Dashboards pattern (v2.51+)
Усі нові дашборди snEco будуються за єдиним стандартом — **standalone HTML page з власним OTP-gate**, не iframe. Окремий whitelist per dashboard (наприклад production-dashboard для Pylyp/Bohdan/Valeriia, finance-dashboard для vg/Pylyp/CFO). Кожен новий дашборд = (a) нова папка `~/snEco/brand/dashboard/<name>/`, (b) новий `block` у Worker SUPPORTED_BLOCKS, (c) окремий KV ключ `wl:<block>`, (d) launcher-картка у sec-dashboard каталозі Brand Bible. Дані у Cloudflare D1 (`sneco-bible`), GitHub Actions cron'и наповнюють. **Sec-dashboard у Brand Bible — це КАТАЛОГ дашбордів** (не сам дашборд), кожен — окрема картка з описом + лінком. Деталі: `memory/project_sneco_dashboards_pattern.md`.
